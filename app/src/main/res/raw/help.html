Help

The LoRaWAN gateway mobile app works with the RAK2287 device connected to the USB port of the phone (tablet).

Run the application.

The gateway on-off switch will become active when the RAK2287 is connected via the USB connector and the right to use the USB device will be granted.

Connect the RAK2287 to a USB port on your phone (tablet).

If necessary, use micro-USB, mini-USB or Type-C adapters.

The gateway consumes a significant amount of power and quickly drains the battery of the phone (tablet). If necessary, use the USB adapter with the charger.

When connecting RAK2287, a dialog box will appear asking for the rights to connect the detected USB device. Confirm that you have granted the right.

Select the desired region from the list.

Turn gateway switch to the on position.

Gateway events are displayed.

Adding end-devices

Click the Devices button.

Click the button with the plus sign (+).

Fill out the form: Device ABP Address, Device ID, Network Key, and Application Key.

Click Save

Add a list of devices

If you have a ready-made list of devices, click Load.

Enter the http:// (https://) address of the service and click the Load button.
